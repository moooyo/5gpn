# Project-maintained native extensions

Each subdirectory is an independently installable `5gpn.io/v1` extension. None
is seeded or enabled by the gateway installer. Operators review and install an
explicit raw manifest URL from the Console.

| Extension | Manifest |
|---|---|
| Apple WLOC Location Override | [`apple-wloc/extension.yaml`](apple-wloc/extension.yaml) |

The manifest format and sandbox API are documented in
[`docs/native-extensions.md`](../docs/native-extensions.md).
